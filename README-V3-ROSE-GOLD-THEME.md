# Merve Yıldırım Beauty — V3 Rose Gold Theme

Bu sürümde ana premium site ile V2 admin onaylı yorum sistemi birleştirildi.

## Yapılan tema değişikliği

- Krem arka plan korunmuştur.
- Ana vurgu rengi gold olarak güncellenmiştir.
- Gül pembesi / rose tonları buton, ikon, gölge, çizgi ve geçişlerde güçlendirilmiştir.
- Layout, sayfa yapısı, metin içerikleri ve yorum sistemi mantığı değiştirilmemiştir.

## Not

Canlı yorum sistemi için `assets/js/review-config.js` içine Supabase URL ve anon key girilmeli, ardından `supabase/reviews-schema.sql` Supabase SQL Editor üzerinden çalıştırılmalıdır.
